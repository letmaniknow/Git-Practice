# Git Practice Scenarios

## Exercise 1: Basic Workflow

1. Create a new file called `hello.txt`
2. Add some content to it
3. Stage and commit the file
4. Push to remote repository

## Exercise 2: Branching Practice

1. Create a new branch called `feature-login`
2. Switch to that branch
3. Create a file called `login.html`
4. Commit the changes
5. Switch back to main
6. Merge the feature branch

## Exercise 3: Conflict Resolution

1. Create two branches with conflicting changes
2. Try to merge them
3. Resolve the conflicts manually
4. Complete the merge

## Exercise 4: History Navigation

1. Use `git log` to view commit history
2. Use `git show <commit-hash>` to see specific changes
3. Use `git diff` to compare changes

## Exercise 5: Undoing Changes

1. Make some changes and don't commit
2. Use `git checkout -- <file>` to undo
3. Make changes, commit, then use `git reset` to undo commit
4. Practice `git revert` to safely undo pushed commits
